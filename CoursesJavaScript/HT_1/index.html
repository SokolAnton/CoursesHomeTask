JavaScript обычно используется как встраиваемый язык для программного доступа к объектам приложений. Наиболее широкое применение находит в браузерах как язык сценариев для придания интерактивности веб-страницам.

Для коментариев используеться так называемый " двойной слэш" - "//" и в коде, чаще всего подсвечиваеться другим цветом, чтобы выделить его от работающего кода. Коментарии бывают разные от: убрать часть кода,для того чтобы посмотреть как работает основной цикл без него, до подписей отдельных частей кода, чтобы разделить для приятного зрительного и поискового использования. Например, для подписи функции, которая считает площадь круга можно сделать Коментарий над этой частью кода: //SCircle

Называть переменные можно по разному, но нужно уметь делать это правильно, используеться CamelCase, где каждое слово имени переменной пишеться с большой буквы, имя переменной также не должно иметь транслита, только английский язык, оно должно четко описывать то, что содержит в себе эта перемнная.
